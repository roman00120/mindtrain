# 📔 Manual de Usuario: Narrativa Workspace

Bienvenido a tu entorno de trabajo profesional para la creación de proyectos cinematográficos y literarios. Este documento explica cómo utilizar cada sección y herramienta de la plataforma.

---

## 1. 🚀 Cásting y Creación de Unidades
Esta es la sección principal para gestionar tus personajes y locaciones.

### ¿Cómo subir imágenes para las cartas?
1. Dirígete a **"Personajes / Lugares"** en el menú lateral.
2. Haz clic en el botón principal **"+ AÑADIR PERSONAJE"** o **"+ AÑADIR ESCENARIO"**.
3. Se abrirá una ventana emergente premium. Verás un botón llamado **"📷 SUBIR IMAGEN"**.
4. Selecciona el archivo de tu ordenador. Verás una vista previa al instante.
5. Al darle a **"GUARDAR"**, se generará una **Tarjeta 3D** con tu imagen aplicada.

### Ficha Detallada
Cada tarjeta tiene un botón de **"ABRIR EXPEDIENTE"**. Al hacer clic, entrarás en una vista profunda donde puedes registrar:
*   **Físico**: Estatura, marcas, complexión.
*   **Psicología**: Traumas, miedos, metas.
*   **Social**: Clase social, educación, religión.

---

## 2. 🏛️ Estructura de la Biblia Narrativa
El menú lateral organiza tu proyecto en 6 módulos maestros:

### 🔹 Base
*   **Formato / Universo**: Define el nombre de tu obra, el género y la duración. Usa el botón de **"GUARDAR"** para asegurar los cambios.
*   **Identidad Visual**: Sube el logo oficial y gestiona la paleta de colores (VFX).
*   **Reglas**: Documenta las leyes físicas de tu universo y las reglas internas para evitar fallos de lógica (agujeros de guión).

### 🔹 Cásting
*   **Personajes / Lugares**: Tu galería visual de unidades con efectos 3D.
*   **Red de Relaciones**: Crea vínculos entre personajes (ej: "A es enemigo de B").

### 🔹 Mundo
*   **Línea de Tiempo**: Registra los hitos cronológicos de tu historia.
*   **Elementos**: Inventario de objetos clave (MacGuffins, props, reliquias).

### 🔹 Desarrollo
*   **Sinopsis / Guión**: Editor de texto en formato Courier para escribir el tratamiento técnico o literario.
*   **Storyboard**: Organización visual de viñetas escena por escena.

### 🔹 Taller (Manuales de Consulta)
Manuales integrados con teoría cinematográfica profesional:
*   **Manual Narrativo**: Explicación de recursos como el Arma de Chejov, Ironía Dramática y Deus Ex Machina.
*   **Manual de Montaje**: Teoría sobre el Efecto Kuleshov y tipos de montaje (Paralelo, Alterno, etc.).

### 🔹 Post
*   Gestión de resultados, festivales y evidencia de producción (Making Of).

---

## 3. 🛠️ Controles Globales (CRUD)
*   **Editar**: Todas las entradas (reglas, objetos, escenas) tienen un botón de `[EDITAR]` para modificar los datos sobre la marcha.
*   **Borrar**: El botón `[BORRAR]` (rojo) elimina permanentemente el elemento tras pedirte confirmación.
*   **Guardar**: En las secciones de texto (Guión, Formato), recuerda pulsar el botón de guardar para actualizar el estado global.

## 💎 Efectos Premium
*   **Tilt 3D**: Las tarjetas de personajes se inclinan físicamente siguiendo el movimiento de tu ratón.
*   **Glow Dinámico**: Un brillo holográfico sigue al cursor dentro de las tarjetas.
*   **Glassmorphism**: Todo el entorno utiliza capas de cristal translúcido para una experiencia visual de alto nivel.
