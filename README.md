# MindTrain

## Descripción
MindTrain es una aplicación web para la gestión y creación de historias, mapas mentales, líneas de tiempo y otros elementos creativos. Incluye funcionalidades de autenticación, suscripciones y publicación de contenido.

## Tecnologías
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: PHP
- **Base de datos**: MySQL
- **Integraciones**: PayPal, Google Login

## Estructura del proyecto
- `assets/`: Recursos estáticos (imágenes, CSS)
- `db/`: Scripts SQL para la base de datos
- `images/`: Imágenes del proyecto
- `js/`: Código JavaScript (aplicación, router, vistas)
- `php/`: Scripts PHP (autenticación, configuración, etc.)

## Instalación
1. Clona el repositorio: `git clone https://github.com/roman00120/mindtrain.git`
2. Configura XAMPP o un servidor web con PHP y MySQL
3. Importa los archivos SQL en `db/` a tu base de datos
4. Configura las credenciales en `php/config.php`
5. Abre `index.html` en tu navegador

## Uso
- Regístrate o inicia sesión
- Crea y gestiona tus historias y elementos creativos
- Publica contenido y administra suscripciones
